  вң… ж–°иҰҸе®ҹиЈ…
  вң… гғўгғҖгғігҒӘгӮўгғјгӮӯгғҶгӮҜгғҒгғЈ
  вң… гғўгғҗгӮӨгғ«гғ»SPA гҒЁгҒ®иҰӘе’ҢжҖ§
  вң… гӮ·гғігғ—гғ«гҒӘе®ҹиЈ…гӮ’жұӮгӮҒгӮӢе ҙеҗҲ

гӮ»гӮӯгғҘгғӘгғҶгӮЈиҖғж…®дәӢй …вҖӢ

й …зӣ®жҺЁеҘЁдәӢй …
зҪІеҗҚгӮўгғ«гӮҙгғӘгӮәгғ RSA-SHA256 д»ҘдёҠгӮ’дҪҝз”Ё
жңүеҠ№жңҹйҷҗзҹӯгҒҸиЁӯе®ҡпјҲж•°еҲҶд»ҘеҶ…пјү
гғӘгғ—гғ¬гӮӨж”»ж’ғеҜҫзӯ–гӮўгӮөгғјгӮ·гғ§гғі ID гӮ’гӮӯгғЈгғғгӮ·гғҘ
XML зҪІеҗҚжӨңиЁјгғ©гғғгғ”гғігӮ°ж”»ж’ғгҒ«жіЁж„Ҹ
иЁјжҳҺжӣёз®ЎзҗҶIdP гҒ®иЁјжҳҺжӣёгӮ’йҒ©еҲҮгҒ«з®ЎзҗҶ
TLSгҒҷгҒ№гҒҰгҒ®йҖҡдҝЎгӮ’жҡ—еҸ·еҢ–

XML зҪІеҗҚгҒ®гғ©гғғгғ”гғігӮ°ж”»ж’ғвҖӢ

SAML гӮўгӮөгғјгӮ·гғ§гғігҒ® XML зҪІеҗҚжӨңиЁјгҒ§гҒҜгҖҒгғ©гғғгғ”гғігӮ°ж”»ж’ғгҒ«жіЁж„ҸгҒҢеҝ…иҰҒгҒ§гҒҷгҖӮ

ж”»ж’ғгӮ·гғҠгғӘгӮӘ:
  1. жӯЈиҰҸгҒ®гӮўгӮөгғјгӮ·гғ§гғігӮ’еӮҚеҸ—
  2. XML ж§ӢйҖ гӮ’ж“ҚдҪңгҒ—гҒҰзҪІеҗҚгӮ’з¶ӯжҢҒгҒ—гҒӨгҒӨеҶ…е®№гӮ’ж”№гҒ–гӮ“
  3. ж”№гҒ–гӮ“гҒ•гӮҢгҒҹгӮўгӮөгғјгӮ·гғ§гғігӮ’йҖҒдҝЎ

еҜҫзӯ–:
  - зҪІеҗҚеҜҫиұЎгҒ®иҰҒзҙ гӮ’еҺіеҜҶгҒ«жҢҮе®ҡ
  - Reference гҒ® URI гӮ’жӨңиЁј
